# CAIXA MESTRE (PWA Offline)

## Requisitos

- Node.js 18+

## Rodar em desenvolvimento

```bash
npm install
npm run dev
```

## Build (para instalar offline)

```bash
npm run build
npm run preview
```

## Assets

Coloque estes arquivos em `public/` (nomes exatos):

- `public/caixa-mestre-logo.png` (sua logo)
- `public/fundo.png` (seu plano de fundo)
